# Yohann Gusso — Portfolio

Pequeno site portfolio desenvolvido com React + TypeScript, Vite e Tailwind CSS para apresentação de projetos, experiência e contato.

Este repositório contém o código do portfolio pessoal de Yohann Gusso — focado em desenvolvimento Full‑Stack e Machine Learning.

## Tecnologias

- React 19 + TypeScript
- Vite (dev server / build)
- Tailwind CSS
- Framer Motion (animações)
- Lucide React (ícones)
